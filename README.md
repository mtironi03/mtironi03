## 🚀 Progetti Principali

### 📦 [BookStore API](https://github.com/mtironi03/BookStoreApi)
**Progettazione e sviluppo di un'API RESTful sicura e ingegnerizzata per la gestione avanzata del catalogo di una libreria.**
* **Stack:** C# (.NET), ASP.NET Core, Entity Framework Core, SQLite, JWT.
* **Dettagli tecnici:** 
  * Realizzazione di un'architettura disaccoppiata basata su DTO (Data Transfer Objects).
  * Sviluppo di un sistema di autenticazione sicuro con token JWT e verifica dell'account tramite OTP via email (MailKit).
  * Implementazione di logiche di validazione complesse (FluentValidation) e gestione chirurgica degli aggiornamenti parziali (PATCH) tramite wrapper personalizzati.
  * Configurazione di una pipeline per la gestione centralizzata delle eccezioni conforme allo standard RFC 7807 (ProblemDetails).

### 🎲 [My Shelfie](https://github.com/mtironi03/My_Shelfie)
**Motore core di backend per la trasposizione digitale dell'omonimo gioco da tavolo, con focus su architettura Object-Oriented, validazione spaziale e testabilità.**
* **Stack:** Java 22, Maven, JUnit 5, MySQL.
* **Dettagli tecnici:**
  * Ingegnerizzazione del dominio di gioco sfruttando ereditarietà e polimorfismo per isolare le logiche degli obiettivi (comuni e personali).
  * Sviluppo di algoritmi di riconoscimento di pattern (pattern matching) su matrici bidimensionali per la scansione della griglia e la verifica delle condizioni di punteggio.
  * Adozione di costrutti Java moderni (es. *switch expression*) per garantire codice pulito (*Clean Code*) e stesura di Unit Test automatizzati per la validazione delle meccaniche.

### 📚 [Biblioteca](https://github.com/mtironi03/Biblioteca)
**Applicazione desktop per la gestione completa di una biblioteca, inclusi utenti, prestiti e catalogo (progetto di gruppo).**
* **Stack:** Java, Java Swing, SQLite, Maven, JUnit.
* **Dettagli tecnici:**
  * Gestione dell'autenticazione con invio di codici via email e autorizzazione basata sui ruoli (Amministratore / Utente).
  * Sviluppo di Unit Test per la validazione e la solidità del codice.
